## 🚀 Minha Jornada na OneBitCode - Full Stack JavaScript

Estou trilhando o caminho para me tornar um desenvolvedor Full Stack com o curso da OneBitCode. Abaixo está a estrutura e os principais tópicos que estou explorando:

### 1. Fundamentos (Fullstack)

- **Link da Seção:** [Fundamentos](https://comunidade.onebitcode.com/c/curso)
- **Principais Tópicos:**
  - Setup do Ambiente de Desenvolvimento
  - HTML 5
  - CSS3
  - JavaScript (Do Básico ao Avançado - Módulos I a VI)
  - Typescript
  - Git e Github

### 2. Backend (Fullstack)

- **Link da Seção:** [Backend](https://comunidade.onebitcode.com/c/curso-backend-javascript)
- **Principais Tópicos:**
  - Introdução ao NodeJS
  - Aplicações Web com NodeJS (Dominando NodeJS)
  - Banco de Dados SQL
  - SQL no Node.js e Prisma ORM
  - MongoDB

### 3. Frontend (Fullstack)

- **Link da Seção:** [Frontend](https://comunidade.onebitcode.com/c/curso-frontend)
- **Principais Tópicos:**
  - CSS Moderno
  - Bootstrap e Sass
  - Fundamentos do React
  - React + TypeScript na Prática (Projeto Kanban)
  - Next.js

### 4. Projetos (Fullstack)

- **Link da Seção:** [Projetos](https://comunidade.onebitcode.com/c/projetos)
- **Principais Projetos:**
  - EAD inspirado no Netflix
  - Agente IA com NodeJS + OpenAI
  - Projeto com Electron - Controlador de Música
